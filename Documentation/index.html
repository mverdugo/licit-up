Getting Started

1. Install recommended version of Node.js

2. Use command line (powershell or terminal) to navigate to project root directory

3. Run following commands, one after another:

npm install
npm start

4. In your browser navigate to http://localhost:3000

5. Thats all! Now you should see your site.

Contact forms

This project uses the service EmailJS to work with contact form. It has versy simple and intuitive set up. All you need to do is create your own account and connect email with service. And then just put your own identification data in template code:

emailjs.send(
  "Service ID",
  "Template ID",
  templateParams,
  "User ID"
)

Deploying

Easiest way to deploy react-based project is Google Firebase, but you can use whichever hosting you like (some steps or setting might be different however). Here is the steps to deploy project in Google Firebase hosting:

Create new project in Google Firebase console.
Run "npm run build" command in your terminal.
Run "npm install -g firebase-tools" command.
Run "firebase login" command and log in to your account in Google
Run "firebase init command":
	- Choose "Hosting: Configure and deploy Firebase Hosting sites"
	- Choose project that you created in previos step
	- Choose "build" as public directory
	- Choose "yes" and configure as a single-page app
	- Choose "no" when asked to overwrite existing index.html
Run "firebase deploy"
Done. Now you get link to deployed project in Google Firebase hosting.

You can also assign your own domain, just check hosting details for steps.

Fonts

By default this template use Poppins font, you can change it in public/index.html:

<link
href="https://fonts.googleapis.com/css?family=Work+Sans:300,400,500,600,700,800&display=swap"
rel="stylesheet"
/>

Icon font Line Awesome added as dependency in package.json. You can use it in the following way:

<i className="las la-phone-volume"></i>    -- all possible icon classes can be found on offsite (link above)
